Vorbereitung

Aktivieren eines Pakets und Aufruf der Hilfe zu einem Paket. Entfernen Sie das “#” vor dem Funktionsaufruf der Hilfe “?ggplot2”, um die Hilfe aufzurufen.

library(ggplot2)             # Aktivieren von Package (nach jedem Start von R)
#?ggplot2                    # Hilfe zu Package

Wertzuweisungen

Übung 1

Weisen Sie der Variablen x den Wert 3 zu und lassen Sie sich anschließend den Wert von x anzeigen.

x <- 3                   # gutes Beispiel einer Wertzuweisung
x                        # Ausgabe von x

Übung 2

Weisen Sie der Variablen y den Wert 1 zu und lassen Sie sich anschließend den Wert von y anzeigen.

1 -> y                   # schlechtes Beispiel einer Wertzuweisung
#y                       # Ausgabe von y

Übung 3

Erstellen Sie die Variable mit dem Namen “klausurnoten”, welche sich aus x und y zusammensetzt. Lassen Sie sich die Variable anzeigen

x <- 3                   # gutes Beispiel einer Wertzuweisung
x                        # Ausgabe von x
1 -> y                   # schlechtes Beispiel einer Wertzuweisung
y                        # Ausgabe von y
klausurnoten <- c(x,y)   # Variable klausurnoten ergibt sich aus x und y
